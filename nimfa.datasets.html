This module lists data sets needed to run Nimfa’s use cases.

Note

Relevant code for data processing is provided within each example. To run an example download the relevant data and put it in nimfa/datasets directory.

Data sets:

  1. Leukemia data set; leukemia gene expressions, leukemia samples, leukemia genes.
  2. Medulloblastoma data set; medulloblastoma gene expressions, medulloblastoma samples, medulloblastoma genes.
  3. CBCL face images database.
  4. ORL face images database.
  5. MovieLens 100k data set.
  6. Medlars data set.
  7. S. cerevisiae MIPS Functional Catalogue data set.
