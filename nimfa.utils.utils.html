Utils (utils.utils)¶

exception nimfa.utils.utils.MFError(msg)¶

Bases: exceptions.Exception

Generic Python exception derived object raised by nimfa library.

This is general purpose exception class, derived from Python common base class for all non-exit exceptions. It is programmatically raised in nimfa library functions when:

#. linear algebra related condition prevents further correct execution of a function; #. user input parameters are ambiguous or not in correct format.
